MobileStoreShop

MobileStoreShop is a Website to save and manage Mobile data in DataBase.

#Installation

Use the package manager [pip] to install Django.
pip install django

#Usage

1-python3 manage.py migrate

2-python3 manage.py createsuperuser  
[OPTIONAL]  to use django admin panel

3-python3 manage.py collectstatic
to use static files(css,fonts)

4-python3 manage.py runserver
